## 简介

记录生活

## 功能

1. 记录

   - 记下随想
   - 随想排序（时间顺序、话题 `#topic`）

2. 支持

   - [markdown 及扩展语法](https://markdown.com.cn/)
   - 多媒体：音频、视频
   - 自定义样式
   - 自定义组件（内置：思维导图、待办事项）

## 使用方法

```bash
# set up
npm install
# 建议使用 cnpm install

# Compiles and hot-reloads for development
npm run chronicle

# see document
npm run doc
```

## 技术栈

1. nodejs
2. vue
3. electron
